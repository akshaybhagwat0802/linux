 ### Filesystem
 ### to create a file 
 ```bash
touch hello.txt
```
### to create multiple files in linux
```bash
touch file{1..5}.txt
```
### to remove the file 
```bash
rm -rvf hello.txt
```
### to remove multiple files 
```bash
rm -rvf file{1..5}.txt
```
### to write in the file
```bash
cat > filename.txt
```
### to read the file
```bash
cat hello.txt
```
### to copy the content of the file
```bash
cat index.html >> hello.html
```
